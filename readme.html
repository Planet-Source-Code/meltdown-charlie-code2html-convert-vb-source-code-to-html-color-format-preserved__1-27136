CODE2HTML

Converts VB source code to color formatted HTML

Code2HTML.bas - is a VB source code module that you can use to format raw source code text into HTML. Code2HTML preserves the tab layout of your source and also attempts to color code it ala the VB IDE.
The code also attempts to optimize the tagging process in order to eliminate repeating font tags where 2 or more reserved words occur together. For instance "Private Function" ...
You may freely use this code in your own projects so long as you acknowledge the origins of it by including a credit somewhere in either your documentation, about box ...
Of course I can't stand over you with a big stick to enforce this, but it's not cool to use without quoting your sources...
